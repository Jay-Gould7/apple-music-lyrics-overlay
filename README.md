# Apple Music Floating Lyrics

一个用于 Windows 桌面的 Apple Music 悬浮歌词小工具。它会读取系统媒体会话中的 Apple Music 播放信息，并在桌面上显示同步歌词、歌曲信息和简洁的播放控制。

## 项目说明

Apple Music Windows 客户端目前没有类似桌面歌词、歌词弹幕或悬浮歌词的展示方式。本工具通过 Windows System Media Transport Controls 读取当前播放歌曲，并用第三方歌词服务匹配 LRC 同步歌词。

需要注意：工具不会直接读取、抓取或爬取 Apple Music 内置歌词，也不会使用 Apple Music 私有接口。歌词来源为 LRCLIB，因此部分歌曲仍可能因为曲名、歌手名、版本信息或歌词库缺失而无法匹配。

## 功能特性

- 通过 Windows SMTC 读取当前 Apple Music 播放信息。
- 支持读取歌曲名、歌手、专辑、播放状态、播放进度和歌曲时长。
- 通过 LRCLIB 获取 LRC 格式同步歌词。
- 精确查询失败后，会自动尝试多种曲名/歌手变体并使用搜索接口兜底。
- 搜索多个候选结果时，会结合歌名、歌手和歌曲时长选择更接近的同步歌词。
- 不再永久缓存“未找到歌词”的空结果，后续可重新尝试匹配。
- 支持本地缓存已成功匹配的歌词，减少重复请求。
- 支持调整歌词同步偏移，范围为 `-2000ms` 到 `+2000ms`。
- 支持调整主歌词颜色、副歌词颜色、字体、字号和窗口大小。
- 支持使用 OpenCC 在原文、简体、繁体之间切换显示。
- 支持透明、无边框、置顶的桌面悬浮窗口。
- 鼠标悬浮在歌词窗口上时显示简洁控制层：
  - 左上角歌曲名和歌手
  - 歌词下方上一首/下一首按钮
  - 右下角播放时间
- 支持通过上一首/下一首按钮控制 Apple Music 播放会话。
- 支持锁定穿透，避免影响鼠标操作其他窗口。
- 支持托盘菜单和全局快捷键恢复交互。

## 使用方式

1. 打开 Apple Music 并播放歌曲。
2. 启动本工具。
3. 工具会自动读取当前播放歌曲并匹配同步歌词。
4. 鼠标悬浮到歌词窗口上时，会显示歌曲信息、播放时间和上一首/下一首按钮。
5. 通过托盘菜单或快捷键打开设置面板后，可以调整颜色、字体、字号、简繁显示和歌词同步偏移。
6. 点击锁定穿透后，窗口会忽略鼠标操作，方便继续使用其他窗口。

## 快捷键

- `Ctrl + Shift + L`：解锁穿透并显示设置面板。
- `Ctrl + Shift + M`：显示或隐藏设置面板。
- `Esc`：关闭设置面板；锁定状态下可尝试恢复交互。

## 托盘菜单

系统托盘菜单提供以下操作：

- `显示窗口`
- `显示设置`
- `解锁穿透`
- `退出`

如果窗口被锁定或看不到设置面板，可以通过托盘菜单或全局快捷键恢复。

## 歌词来源与限制

- 工具不会读取 Apple Music 内置歌词。
- 工具使用 LRCLIB 进行第三方同步歌词匹配。
- Apple Music 官方公开接口不提供歌词正文或逐行时间轴。
- 部分歌曲可能仍然检索不到同步歌词。
- 不同版本的歌曲可能存在时间轴偏移，可通过同步偏移滑杆调整。
- Apple Music 返回的歌手字段有时会混入专辑或单曲信息，可能影响匹配准确率。
- 网络不可用或 LRCLIB 服务不可达时，歌词加载会失败。

## 常见问题

### 歌词出现太慢或太快

打开设置面板，调整同步偏移：

- 歌词出现慢：增加“提前”数值。
- 歌词出现早：调到“延后”方向。

默认偏移为 `提前 650ms`。

### 原文、简体、繁体怎么选

设置面板里的 `原` / `简` / `繁` 小按钮用于切换歌词显示：

- `原`：按歌词源原文显示。
- `简`：使用 OpenCC 转为简体显示。
- `繁`：使用 OpenCC 转为繁体显示。

如果使用 `原` 或 `繁` 时选择了部分装饰字体，可能会出现个别繁体字不兼容、字形不统一或回退到其他字体的情况。建议优先使用 `简` 模式，或者选择思源黑体、思源宋体、微软雅黑、圆体等简繁覆盖更完整的字体。

### 鼠标悬浮时显示的信息太多

普通悬浮只显示歌曲名、歌手、播放时间和上一首/下一首按钮。颜色、字体、字号、简繁切换和同步偏移等设置只会在打开设置面板时显示。

### 锁定后无法点击窗口

这是锁定穿透模式的预期行为。可以使用以下方式恢复：

```text
Ctrl + Shift + L
```

或从托盘菜单选择 `解锁穿透`。

### 检索不到歌词

可能原因：

- LRCLIB 没有该歌曲歌词。
- 当前歌曲是 Live、Remix、伴奏或特殊版本。
- Apple Music 返回的曲名或歌手名与歌词库记录不一致。
- 歌曲 metadata 与歌词库记录时长差异较大。

工具已经会自动尝试精确查询、清理后的曲名/歌手查询以及搜索兜底，但无法保证所有歌曲都能匹配。如果歌词库本身没有该歌曲歌词，需要依赖本地歌词文件等兜底方式。
